# 删除共享<a name="sfs_02_0027"></a>

## 功能介绍<a name="s756a340c47e0467f98ab7c716a729342"></a>

删除共享。

>![](public_sys-resources/icon-note.gif) **说明：**   
>该接口为异步接口，返回为202只是表示接口下发接收成功，后续可以通过[查询共享详细信息](查询共享详细信息.md)查询共享，判断删除共享是否完成以及是否成功。
