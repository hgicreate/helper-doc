# 批量删除标签<a name="sfs_02_0042"></a>

## 功能介绍<a name="section10684447163819"></a>

为指定共享批量删除标签。

此接口为幂等接口：如果共享上不存在待删除的标签，则默认删除成功。

## URI<a name="section1665327514513"></a>
