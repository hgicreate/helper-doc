# 删除标签<a name="sfs_02_0038"></a>

## 功能介绍<a name="section10684447163819"></a>

指定共享删除一个标签。

>![](public_sys-resources/icon-note.gif) **说明：**   
>当共享中不存在指定要删除的key时，接口调用将会返回404错误。  

## URI<a name="section1665327514513"></a>
